# Assignment 1 - C++ and OpenMP
## Задания
### Task 1
- Динамический массив из 50 000 чисел
- Вычисление среднего значения

### Task 2
- Последовательный поиск минимума и максимума в массиве из 1 000 000 чисел
- Замер времени выполнения

### Task 3
- Параллельный поиск минимума и максимума в массиве из Task 2 с использованием OpenMP
- Сравнение времени с последовательным алгоритмом

### Task 4
- Вычисление среднего значения массива из 5 000 000 чисел
- Последовательно и параллельно с OpenMP (reduction)
- Сравнение времени выполнения

## Инструкция по запуску
- Компилируйте любой `.cpp` файл через Visual Studio или любой компилятор C++
- Для параллельных заданий требуется **подключение OpenMP** (флаг `-fopenmp` для g++, включить поддержку OpenMP в Visual Studio)
