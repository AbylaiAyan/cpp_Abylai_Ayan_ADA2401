# Практическая работа №3
**Тема:** Динамическая память, указатели и параллельное вычисление среднего значения массива с использованием OpenMP

---

## Описание работы
В рамках данной практической работы была реализована программа на языке C++, демонстрирующая работу с динамической памятью и указателями, а также применение параллельных вычислений на CPU с использованием технологии OpenMP.

Программа создаёт динамический массив заданного размера, заполняет его случайными числами и вычисляет среднее значение элементов массива двумя способами: последовательным и параллельным. Для обоих вариантов производится измерение времени выполнения и их сравнение.

---

## Функциональность программы
Программа выполняет следующие действия:

1. Считывает с клавиатуры размер массива N
2. Выделяет динамическую память под массив с помощью оператора new
3. Заполняет массив случайными целыми числами в диапазоне от 1 до 100
4. Вычисляет среднее значение элементов массива:
   - последовательным способом
   - параллельным способом с использованием OpenMP (reduction)
5. Измеряет и сравнивает время выполнения обоих способов
6. Выводит:
   - среднее значение
   - время выполнения
   - количество логических ядер процессора
   - количество реально использованных потоков OpenMP
7. Освобождает выделенную динамическую память с помощью delete[]

---

## Используемые технологии
- Язык программирования: C++
- Параллельное программирование: OpenMP
- Генерация случайных чисел: <random>
- Измерение времени выполнения: <chrono>

## Как запустить программу

### В Visual Studio
1. Откройте проект и добавьте файл task3.cpp
2. Включите поддержку OpenMP:
   Project → Properties → C/C++ → Language → OpenMP Support → Yes
3. Соберите и запустите проект
4. Введите размер массива при запуске программы

### Компиляция через g++
```bash
g++ -fopenmp -O2 task3.cpp -o task3
